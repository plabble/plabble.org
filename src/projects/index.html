CARD 'plabble' _m Plabble Chat Protocol (PCP) is an end-to-end encrypted channel-based chat protocol. Our main goal is to ensure authenticity and confidentiality by design, while preserving availability using (self-hosted) servers. Chatting with Plabble is anonymous because the identity of a user is not linked to any personal data, but to a cryptographic keypair. To stay safe users are allowed to use multiple keypairs. **[ [repo](https://github.com/plabble) [docs](https://plabble.github.io) ]** by maurictg, m1sth1os and job79 _m ENDCARD CARD 'aweb' _m Aweb is a modular static website generator written in posix shell.
